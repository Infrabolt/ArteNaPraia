# ArteNaPraia
Projeto de testes (manuais e automatizados) aplicado a um sistema de e-commerce fictício de venda de artes chamado "Praia". O foco está na validação de cenários negativos (Unhappy Path), garantindo a robustez do sistema diante de falhas, exceções e comportamentos inesperados. Desenvolvido com Robot Framework e organizado sob metodologia Kanban.
